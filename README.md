# Domino Projekt

## Systemanforderungen

Das Projekt ist Maven-basiert und wurde zuletzt erfolgreich mit der Maven-Version 3.13.0 und der Java-Version 21.0.5 getestet.
Das JavaFX-Plugin verwendet die JavaFX-Version 21.0.6.

## Installation und Inbetriebnahme

Klonen Sie das Repository mit Git oder laden Sie die ZIP-Datei direkt von GitHub herunter:

1. **Falls Repository klonen:**
   `git clone https://github.com/rudnevs20/Domino.git`
2. **Anwendung ausführen:**
   Navigieren Sie in das Projektverzeichnis und führen Sie `mvn clean javafx:run` aus.

**Ressourcen:**
Zusätzliche Dokumentationen, Videos und ZIP-Dateien sind auf [Google Drive](https://drive.google.com/drive/folders/1Q47xA-l4OwCMcCsILQ4bufRozPd5Ib_p?usp=sharing) verfügbar.
